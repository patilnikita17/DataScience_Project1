# Customer Churn Prediction
## Overview
This project predicts customer churn using machine learning techniques...
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
## Dataset
Telco Customer Churn Dataset containing customer demographics, services, and billing information.
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
## Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
## Models Used
- Logistic Regression
- Decision Tree Classifier
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
## Evaluation Metrics
- Accuracy
- ROC-AUC
- Confusion Matrix
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
## Key Insights
- Customers with low tenure are more likely to churn
- Month-to-month contracts show higher churn
- Long-term contracts reduce churn significantly
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
## Business Impact
This model helps companies identify high-risk customers early and take retention actions.
